# Brightness
## brightness.getLevel() ##
#### Syntax
```js
var brightness = brightness.getLevel();
```

#### Return value
A `Number` between 0 and 1 with the current brightness value


## brightness.setLevel() ##
#### Syntax
```js
brightness.setLevel(level);
```

#### Parameters
`level`: A `Number` between 0 and 1 to set the brightness level
