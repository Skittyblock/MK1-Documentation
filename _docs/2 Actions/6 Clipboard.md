# Clipboard
## clipboard.get() ##
#### Syntax
```js
var clipboard = clipboard.get();
```

#### Return value
A `String` containing the current clipboard text


## clipboard.set() ##
#### Syntax
```js
clipboard.set(string);
```

#### Parameters
`string`: A `String` that you want to be copied to the clipboard
