# Low Power Mode
## lpm.isEnabled() ##
#### Syntax
```js
var enabled = lpm.isEnabled();
```

#### Return value
A `Boolean` representing the current enabled status of low power mode


## lpm.setEnabled() ##
#### Syntax
```js
lpm.setEnabled(enabled);
```

#### Parameters
`enabled`: A `Boolean` to set low power mode enabled status
