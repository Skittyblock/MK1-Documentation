# Flashlight
## flashlight.getLevel() ##
#### Syntax
```js
var level = flashlight.getLevel();
```

#### Return value
A `Number` between 0 and 1 representing the current flashlight level


## flashlight.setLevel() ##
#### Syntax
```js
flashlight.setLevel(level);
```

#### Parameters
`level`: A `Number` between 0 and 1 to set the flashlight level
