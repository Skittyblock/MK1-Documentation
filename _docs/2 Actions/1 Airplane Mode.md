# Airplane Mode
## airplaneMode.isEnabled()
#### Syntax
```js
var enabled = airplaneMode.isEnabled();
```

#### Return value
A `Boolean` with the status of airplane mode.

## airplaneMode.setEnabled()
#### Syntax
```js
airplaneMode.setEnabled(enabled);
```

#### Parameters
`enabled`: A `Boolean` to set airplane mode status
