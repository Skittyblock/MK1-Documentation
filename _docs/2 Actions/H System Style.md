# System Style
## systemStyle.isDark() ##
#### Syntax
```js
var darkMode = systemStyle.isDark();
```

#### Return value
A `Boolean` representing the status of dark mode


## systemStyle.toggle() ##
#### Syntax
```js
systemStyle.toggle();
```
