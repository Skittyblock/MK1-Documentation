# URL Schemes
The MK1 application implements URL schemes that allow you to run scripts and activate triggers.
Format: `mk1://` + action + argument

### Run script
`mk1://runScript/name`
`name`: script name

### Activate trigger
`mk1://runTrigger/name`
`name`: trigger name
